Segundo Encontro do GURUSC

O Grupo de Usuários Ruby de Santa Catarina está promovendo o seu segundo encontro em Florianópolis/SC.

O evento tem apoio do @SenacTI que disponibilizou o local para a realização do encontro.

O objetivo do encontro é possibilitar a troca de experiência da comunidade de desenvolvimento de software de Santa Catarina, não se limitando as tecnologias envolvidas na linguagem Ruby.

Além disso, é uma ótima oportunidade para fazer networking com profissionais da região e também de fora do estado.

  • Quando: 01 de outubro de 2011 das 10h às 15h

  • Onde: Florianópolis/SC - Auditório da Fecomércio

  • Sugestão de hotel: Hotel Valerim

Aplicações Rails Dinâmicas usando Backbone.js

O uso pesado de JavaScript em aplicações web se tornou muito comum. No mundo Ruby on Rails não é diferente. Apesar de existirem diversos metodos para auxiliar no desenvolvimento dessas aplicações mais interativas, ainda existem uma mescla grande de código Ruby com JavaScript.

Nessa palestra Rafael falará sobre Backbone.js, um framework JavaScript MVC, que propõe uma separação simples de código JavaScript com o código do backend. Rafael irá mostrar os conceitos do Backbone.js mesclando teoria e código e concluindo com exemplos de integração de Ruby on Rails e Backbone

  • Palestrante:Rafael Felix
